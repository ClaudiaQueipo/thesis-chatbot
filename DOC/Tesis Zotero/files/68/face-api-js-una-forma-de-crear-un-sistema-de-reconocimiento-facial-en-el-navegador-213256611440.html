face-api.js: una forma de crear un sistema de reconocimiento facial en el navegador.

Un sistema de reconocimiento facial es una tecnología capaz de comparar un rostro humano a partir de una imagen digital o un fotograma de video con una base de datos de rostros, normalmente empleado para autenticar a los usuarios a través de servicios de verificación de identidad, funciona identificando y midiendo los rasgos faciales de una imagen determinada.

Este artículo se centrará en cómo implementar un sistema de reconocimiento facial en el navegador usando JavaScript. Para implementar esto, usaremos la biblioteca de JavaScript face-api .

¿Qué es face-api.js?

face-api es una biblioteca de JavaScript creada por Vincent Mühler., para detectar rostros a través del navegador. Está construido sobre la API central de tensorflow.js. Es compatible con la detección de rostros, el reconocimiento de rostros, la expresión de rostros, la edad y la detección de género.

Cargar los modelos

El siguiente sería cargar el modelo. loadSsdMobilenetv1Modelse utiliza para cargar el ssdmodelo de detección de objetos, loadFaceLandmarkModelpara detectar el punto de referencia facial, loadFaceRecognitionModelpara detectar el reconocimiento facial y loadFaceExpressionModelpara detectar la expresión facial

const MODEL_URL = '/models' //model directory
await faceapi.loadSsdMobilenetv1Model(MODEL_URL)
await faceapi.loadFaceLandmarkModel(MODEL_URL) // model to detect face landmark
await faceapi.loadFaceRecognitionModel(MODEL_URL) //model to Recognise Face
await faceapi.loadFaceExpressionModel(MODEL_URL) //model to detect face expression

Reconocimiento facial

Ahora intentemos reconocer la cara de la imagen de arriba.

Crea una matriz labelque contenga los nombres de los personajes de la imagen. Nombra la imagen de referencia igual que la etiqueta. fetchImagepodría usarse para cargar imágenes desde los directorios.

Detecte una sola cara y calcule la descripción de la cara para cada imagen y almacénela junto con la etiqueta respectiva con LabeledFaceDescriptors.

const labels = ['ross', 'rachel', 'chandler', 'monica', 'phoebe', 'joey']
const labeledFaceDescriptors = await Promise.all(
labels.map(async label => {
const imgUrl = `images/${label}.jpg`
const img = await faceapi.fetchImage(imgUrl)
const faceDescription = await faceapi.detectSingleFace(img).withFaceLandmarks().withFaceDescriptor()
if (!faceDescription) {
throw new Error(`no faces detected for ${label}`)
}
const faceDescriptors = [faceDescription.descriptor]
return new faceapi.LabeledFaceDescriptors(label, faceDescriptors)
})
);
view raw detect.js hosted with ❤ by GitHub

Haga coincidir la cara de entrada con la descripción de la cara de referencia con la Distancia euclidiana . Se puede establecer el valor umbral para la distancia euclidiana. Al establecer el valor de umbral, definimos que la imagen de referencia a igualar debe tener como máximo el valor de umbral específico.

faceapi.FaceMatcher()toma el descriptor de la característica de referencia y el umbral para inicializar el objeto para calcular la distancia euclidiana y aplicar el umbral. Esto se hace a través de la faceMatcher.findBestMatch()función recorriendo todas las características detectadas para cada rostro en la imagen de entrada.

const threshold = 0.6
const faceMatcher = new faceapi.FaceMatcher(labeledFaceDescriptors, threshold)
const results = faceDescriptions.map(fd => faceMatcher.findBestMatch(fd.descriptor))
//label images
results.forEach((bestMatch, i) => {
const box = faceDescriptions[i].detection.box
const text = bestMatch.toString()
const drawBox = new faceapi.draw.DrawBox(box, { label: text })
drawBox.draw(canvas)
})
